## How to Use This Doc

1. Estimate the effort category for each feature
2. Add up all features
3. Apply multiplier based on complexity
4. Add buffer (10-15%) for revisions
5. Convert to USD for international clients (1.5-2x)

## Project Multipliers

Use to adjust base price:

| Factor | Multiplier | When |
|--------|------------|------|
| Urgency | 1.25-1.5 | Needed in < 1 week |
| Complexity | 1.25-1.5 | Heavy logic/complex integrations |
| Research needed | 1.25 | New domain/tech stack |
| Ongoing support | 0.9 | Multi-month retainer (discount for volume) |

## USD Conversion Guide

For international clients:
- **Multiply INR by 1.5** (conservative)
- **Multiply INR by 2** (standard)

Example: ₹2,40,000 × 1.5 = **$3,600**
Example: ₹2,40,000 × 2 = **$4,800**

Adjust based on client region and negotiation.
